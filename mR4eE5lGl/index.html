python从入门到入土

2024-03-03

注: python 和 vscode 安装都是免费的，如果你遇到收费的请立刻删除。

安装

python 安装

liunx：都用 liunx，还不会安装 py 嘛？？？
mac：我没用过，不讨论。
window：往下看。

  1. 打开 python 官网https://www.python.org/
  2. 点击 Download。 如果你没有科学上网的话，此过程将耗费你 1 到 5 分钟，或者更多。
  3. 然后点击黄色的 windows。
  4. 然后点击这个搜索框，输入 3.10.11 。如果你是 win7 的用户，推荐升级 win10 或者 google 一下 win7 安装 py 和 vscode 教程
  5. 往下翻，然后点击 3.10.11 这个。
  6. 进入这个页面然后往下翻，翻到最底下。下载 Windows installer 64-bit
  7. 慢慢等他下载吧，毕竟是国外的网站，下载速度懂得都懂。
  8. 下载好了双击安装包。因为我已经装过 3.10.11 了，所以用 3.11.6 的做演示
  9. 这个界面可能会有一个突破 MAX_PATH 的那个选项，看你心情选不选都行因该。我推荐勾选上，解除一下限制。

    来自这个博客的示意图。

  10. 然后 Close。
  11. 恭喜你装好了。
  12. 打开 cmd，输入 python -V 如果出现 Python 3.10.11 就证明装好了。
